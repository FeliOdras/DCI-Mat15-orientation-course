How it all begann

It was in 1999 when I "built" my first website. It was kind of crappy, built with a WYSIWYG-Editor and it only worked in Internet Explorer. A few years later I came to a place called "c-base" (the people who run this place pretend to build a crashed space station, some of them might really believe it) and there I met some cool guys who really knew how to build a website. One of them showed me the German HTML-Reference project selfhtml.org. 'If you want to build a website, read this', he said. So I did and so it came that I fell in love with code.
